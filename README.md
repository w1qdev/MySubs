# MySubs

MySubs — pet‑проект для монетизации закрытых сообществ. Сервис позволяет создавать и управлять подписками, пользователями и интеграциями для закрытого контента (например, доступ к каналам, группам или материалам по подписке).

Цели проекта:

- Реализовать управление подписками и платёжными потоками.
- Поддержать разные способы аутентификации и интеграции (Telegram, Discord и т.д.).
- Обеспечить безопасное хранение пользователей и подписок (валидация, ограничение доступа).

Структура репозитория:

- `backend/` — NestJS API (модули: `auth`, `users`, `subscriptions`, `payments`, интеграции и т.д.).
- `frontend/` — React + Vite фронтенд-приложение с интерфейсом для пользователей и админки.

Быстрый старт (требуется Node.js и пакетный менеджер):

1. Установить зависимости для backend и frontend:

```bash
cd backend
pnpm install

cd ../frontend
pnpm install
```

2. Настроить переменные окружения:

- Создайте `.env` в `backend/` на основе `.env.example` (если есть) и заполните необходимые переменные: `PORT`, `DATABASE_URL`, `JWT_SECRET`, данные для платёжных провайдеров и интеграций.

3. Запустить в режиме разработки:

```bash
# backend
cd backend
pnpm run start:dev

# frontend (новая вкладка)
cd frontend
pnpm run dev
```
